public class WRImageItem
extends ImageDisplayItem
implements Alignable, Observer
Class WRImageItem is the display item for all images that appear in HotJava documents. It handles alignment queries, delayed image loading, image caching and following hypertext links if the image is part of an anchor. Instances of this class are observers of instances of ImageHandle. That way this display item will get notified when the ImageHandle finishes reading an image across the net.
