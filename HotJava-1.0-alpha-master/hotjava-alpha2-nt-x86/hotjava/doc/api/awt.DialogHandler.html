Interface awt.DialogHandler

public interface DialogHandler
extends Object
DialogHandler is an interface used by users of the MessageDialog class. It defines three callbacks which are invoked by MessageDialog in response to clicking on one of the three buttons, "ok", "cancel", and "help".
