public class DocumentManager

Class DocumentManager manages multiple html documents. It maintains a list of visited documents, and fetches ones which are not currently known. Instances of DocumentManager are created by WRWindow to manage the documents of that window. Documents are fetched in the background, and multiple documents may be fetched at the same time.
