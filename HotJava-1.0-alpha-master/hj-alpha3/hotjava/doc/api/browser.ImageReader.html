public class ImageReader
extends Thread
Class ImageReader is a thread which sits around waiting for requests to fetch images in behalf of ImageHandles. It performs fetches in background threads, and allows up to ImageReader.simultaneousFetcherLimit requests at the same time. ImageReader is created and managed by instances of WRWindow.
