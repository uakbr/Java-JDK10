  public Toggle(String pLabel,

                Container pParent,
                RadioGroup group,
                boolean state)
Constructs a Toggle.
Parameters:
pLabel - is the label of this toggle button.
pName - is the name of this Toggle.
pParent - is the window to contain this Toggle.
group - is the RadioGroup this Toggle is in. If not null, then this Toggle becomes a radio button which means only one Toggle in a RadioGroup may be set.
state - is the initial state of this Toggle.
