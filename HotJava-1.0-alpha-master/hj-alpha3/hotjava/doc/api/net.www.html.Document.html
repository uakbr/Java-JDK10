public class Document

net.ww.html.Document is a class which holds an html document. An html document is parsed text, a vector of tags, and optional html source text. Document is used by html.Parser to store parsed documents, but it is also used to help programatically generate html documents. E.g., the ftp protocol handler might generate an html document out of a directory listing.

htmlSource
